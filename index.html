FireDoc - Um projeto que está sendo desenvolvido juntamente a dois colegas de faculdade desde do primeiro período estamos em total dedicação a esse projeto.
O que é a FireDoc afinal?
O nome FireDoc significa fogo nos documentos, uma joga de marketing para que possamos eliminar todos os documentos físicos e utilizarem apenas documentos eletrônicos em nossa plataforma, o nome que foi sugerido por um de nossos integrantes desse grupo o projeto visa em um sistema GED(Gerenciamento Eletrônico de Documentos). Nesse projeto inicialmente foi feito uma pesquisa de campo e logo a após foi desenvolvido os diagramas. Atualmente estamos desenvolvendo o banco de dados em SqlServer que irá receber os dados de nosso GED, e estamos em processo para iniciarmos o desenvolvido da interface gráfica web.
